#### 관계형 DB

관계형 DB는 아주 빨라야 함.
주로 이제 관계를 찾기 위해서 사용하는 Key를 foreign Key (외래키)라고 부름

**Primary Key**
- 각각의 테이블마다 고유한 Key가 있고
- A테이블에서 B테이블의 key를 참조할 때 foreign Key라고 함.
- StudentID : Primary Key
  - Student Table의 수강과목 ID => Foreign Key
  - 그래서 mapping table을 사용함. (귀찮으니 알아서 알아보도록)
  

  ### 예시
```C#
class Hero
{
    public List<Skill> _skillList;
    public List<Item> _itemList;
    public string _name;
    public int _level;
    public List<Stat> _statList;
}
```
